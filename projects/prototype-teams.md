# Prototype Project Teams — Spring 2026

Teams were formed using survey data covering skills, interests, and teammate preferences, then optimized using [gruepr](https://gruepr.com) (a free, open-source team formation tool).

Topics below are drawn from each team's submitted Project Proposal.

## Teams
